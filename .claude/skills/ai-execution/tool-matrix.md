# AI Tool Matrix — Content Type → Best Tool

## Nguyên tắc chọn tool
1. **Free first:** Ưu tiên tool miễn phí nếu output đủ tốt
2. **Claude cho text:** `/copywrite` skill xử lý tốt hơn ChatGPT cho content tiếng Việt có brand voice
3. **Canva cho visual nhanh:** Canva AI đủ tốt cho 80% social graphics — chỉ dùng Midjourney khi cần ảnh quality cao
4. **Batch similar tasks:** Làm tất cả caption cùng lúc, tất cả visual cùng lúc — tiết kiệm context switching
