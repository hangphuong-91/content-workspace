# Blog Page Template — Notion Long-form Content

**Dùng khi:** Tạo bài blog, LinkedIn Article, hoặc bất kỳ bài Long-form nào trong Notion  
**Cách dùng:** Copy toàn bộ section "TEMPLATE" bên dưới → paste vào page body của row Notion → điền nội dung

---

## Cách Áp Dụng Trong Notion

### Cách 1: Template Button (Khuyến Cáo)
```
1. Mở Database → "..." → "Edit templates"
2. Click "+ New template"
3. Đặt tên: "Blog / Long-form Post"
4. Paste nội dung template bên dưới vào page body
5. Save → Mỗi lần tạo row mới: chọn template này
```

### Cách 2: Manual Copy-Paste
```
1. Tạo row mới trong Content Calendar
2. Click tên row để mở page
3. Copy từ "==== BẮT ĐẦU TEMPLATE ====" đến cuối
4. Paste vào page body → xóa dòng hướng dẫn → điền nội dung
```

### Chèn Ảnh Trong Notion
```
Cách 1: Gõ "/image" → Upload from computer hoặc paste URL
Cách 2: Kéo thả file ảnh trực tiếp vào page
Cách 3: Paste link từ Google Drive (share → copy link)
Cách 4: Paste link export từ Canva

Vị trí đặt ảnh:
- Ảnh đầu bài (Hero image): ngay dưới tiêu đề
- Ảnh giữa bài: sau mỗi H2 section quan trọng
- Ảnh thông tin (Infographic): cuối bài trước CTA
```

## Checklist Trước Khi Publish

### Nội Dung
- [ ] Tiêu đề có từ khóa chính, hấp dẫn, < 60 ký tự
- [ ] Meta description 150–160 ký tự, có từ khóa
- [ ] Câu thu hút (Hook) đầu bài đủ mạnh (đặt câu hỏi / con số / tuyên bố mạnh)
- [ ] Mỗi H2 có từ khóa phụ hoặc liên quan
- [ ] Không có đoạn văn > 5 câu (tối ưu mobile)
- [ ] Có ít nhất 1 blockquote hoặc callout nổi bật
- [ ] Lời kêu gọi hành động (CTA) rõ ràng, 1 CTA chính duy nhất
- [ ] Compliance check (nếu ngành y tế: không dùng "chữa khỏi", dùng "hỗ trợ cải thiện")

### Hình Ảnh
- [ ] Hero image đầu bài (1.200×628px cho blog, 1.080×1.080px cho LinkedIn)
- [ ] Ít nhất 1 ảnh giữa bài mỗi 400–500 từ
- [ ] Ảnh thông tin (Infographic) cuối bài (nếu có)
- [ ] Tất cả ảnh có alt text (mô tả ngắn)
- [ ] Featured Image đã điền vào property Notion
- [ ] Has Images = ✅ trong Notion

### SEO (Cho Blog)
- [ ] Từ khóa chính xuất hiện trong: Tiêu đề, H2 đầu tiên, đoạn đầu, kết luận
- [ ] Internal links: ít nhất 1 link tới bài khác cùng website
- [ ] External links: 1–2 link nguồn uy tín (nếu dẫn số liệu)
- [ ] URL slug: ngắn, có từ khóa, không dấu tiếng Việt

### Kỹ Thuật
- [ ] Word Count đã điền vào Notion property
- [ ] Status → ✅ Ready
- [ ] Assign cho người review

## Ví Dụ Áp Dụng — Bác Sĩ Sản Phụ Khoa

```
Tiêu đề SEO: Thai Kỳ Tuần 28: 5 Điều Bé Đang Làm Mà Mẹ Chưa Biết
Meta: Tuần 28 thai kỳ, bé nặng ~1,2kg và đang phát triển mạnh. 
      Bác sĩ chia sẻ 5 điều thú vị về sự phát triển của bé tuần này.
Từ khóa chính: thai kỳ tuần 28
Từ khóa phụ: sự phát triển thai nhi, bé tuần 28, thai 28 tuần
Persona: Mẹ bầu lần đầu, 25–35 tuổi, lo lắng về sự phát triển của bé
Funnel: Awareness → Consideration
Pillar: Giáo dục thai kỳ
Thời gian đọc: ~4 phút
Word Count: ~800 từ
```
