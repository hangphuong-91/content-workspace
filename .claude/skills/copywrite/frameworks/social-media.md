# Copywriting Frameworks — Social Media

## Facebook / Instagram Post

### Framework: Hook → Pain/Story → Solution → Social Proof → CTA

**Cấu trúc:**
```
[HOOK — dòng 1 phải dừng scroll. Kết thúc bằng "..." để user bấm See more]

[Line break]

[Pain point hoặc story ngắn — 2–3 câu. User phải thấy mình trong đó]

[Line break]

[Solution / Insight / Giá trị cốt lõi — rõ ràng, không vòng vo]

[Line break]

[Social proof: "X bệnh nhân hỏi mình điều này" / "Mình thấy điều này mỗi ngày" / Số liệu]

[Line break]

👉 [CTA — 1 hành động duy nhất, rõ ràng]
#hashtag1 #hashtag2 #hashtag3
```

**Hook formulas hiệu quả nhất:**
- "Điều mà không bác sĩ nào nói với bạn về [topic]..."
- "[Con số] lý do tại sao [belief phổ biến] là sai..."
- "Nếu bạn đang [situation], đọc bài này trước khi..."
- "Hôm nay có bệnh nhân hỏi mình một câu khiến mình suy nghĩ mãi..."
- "Mình đã sai về [điều này] suốt [X năm]..."

**Quy tắc vàng Facebook:**
- Dòng 1: Hook — quyết định 80% reach
- Không dùng "Tag bạn bè" hoặc "Like nếu đồng ý" (Facebook penalize)
- Reply comment trong 60 phút đầu → boost distribution đáng kể
- Line break sau mỗi 2–3 câu (dễ đọc trên mobile)

## TikTok / Reels Caption

Caption TikTok ngắn — chỉ cần 1–2 dòng vì người xem tập trung vào video:
```
[Hook câu hỏi hoặc statement kích thích comment]
[Hashtag: 3–5 mix broad + niche]
```

**TikTok Script Structure:**
```
[0–3s] HOOK VISUAL + AUDIO: [Hành động + câu nói đầu tiên gây tò mò]
[3–8s] TENSION: [Đặt câu hỏi / tạo conflict / setup]
[8–40s] VALUE: [Nội dung chính — facts, tips, story]
[40–60s] CTA: [Comment / Follow / Link bio]
[Text overlay cuối: CTA + username]
```

**Hook formulas cho TikTok:**
- "POV: Bạn vừa hỏi bác sĩ câu này..." (relatability)
- "Sự thật mà Google không nói với bạn về [topic]" (curiosity)
- "[Số] giây và bạn sẽ biết [outcome]" (time commitment thấp)
- Bắt đầu bằng hành động — không giới thiệu, vào thẳng vấn đề

## LinkedIn Post

### Framework: Insight Hook → Problem → Framework → CTA Professional

```
[Hook: 1 câu insight bất ngờ hoặc ngược chiều số đông]

[Dòng trắng]

[Context: Tại sao điều này quan trọng — 2–3 câu]

[Dòng trắng]

[Framework: Dùng số hoặc bullet]
1. [Điểm 1]
2. [Điểm 2]
3. [Điểm 3]

[Dòng trắng]

[Kết: Bài học / Quan điểm cá nhân / Câu hỏi mở]

[CTA chuyên nghiệp: "Bạn nghĩ sao? Comment bên dưới."]
```

**LinkedIn-specific rules:**
- Dòng 1 phải khiến người ta click "See more" → đừng kết thúc bằng dấu chấm
- Professional tone nhưng phải có personal touch (kể chuyện thật)
- Không dùng hashtag nhiều — 3–5 là đủ, để cuối bài
- Best performing: story từ kinh nghiệm thực tế + lesson rõ ràng
