## 👥 The 2 Agents

### Agent 1: Content Planner (CMO Role)

**Owns:** Strategy, validation, measurement, direction

```
QUARTERLY CYCLE:
├─ W19 (before W20): Research + planning (5 phases, 56 pieces, 6 hypotheses)
├─ W20-W21: Launch + monitor H1 measurement (Reels vs static)
├─ W22-W24: Scale + H2 decision (Luật content validation)
└─ W25-W30: Optimize + H3-H6 testing + M3 planning

WEEKLY CADENCE:
├─ Monday-Wednesday: Develop W[n+1] direction brief (3-4 pieces)
├─ Friday: Receive Execution snapshot → run Sume-Kit synthesis
├─ Friday-Saturday: Hypothesis analysis + pivot decision (if needed)
└─ Sunday: Finalize W[n+1] direction for Monday handoff

TOOLS: Notion (KPI tracking), GitHub (version control), Think-Coach (validation), Sume-Kit (synthesis)

ESCALATION: Owns final decisions on pivots, compliance, resource conflicts
```

### Agent 2: Content Execution (Operator Role)

**Owns:** Production, publishing, measurement, feedback

```
WEEKLY CYCLE:
├─ Monday 09:00: Receive W[n] direction from Planner
├─ Monday-Wednesday: Copywrite (3 pieces) + Design (1-2 pieces)
├─ Tuesday-Wednesday: BS Thư filming (2 shoots, 4 videos)
├─ Wednesday-Friday: Publish 4 pieces across channels (staggered timing)
├─ Friday: Collect 7-day metrics → fill Notion → snapshot to Planner
└─ Friday 18:00: Send W[n] snapshot + learnings to Planner

DAILY TASKS:
├─ Copywrite: /copywrite tiktok [brief] → 3 variations per piece
├─ Design: /visual creative [brief] → Canva link + PNG export
├─ Production: Shoot 2 videos (60-90s) + edit in CapCut + remove watermarks
├─ Publishing: Post 3-4 pieces (timing per algorithm + persona)
├─ Measurement: Track reach/ER%/shares + flag H1-H3 anomalies

COMPLIANCE GATE: 5-item checklist before every post (no cures, patient consent, etc.)

TOOLS: Claude /copywrite, Canva /visual creative, CapCut, Meta Business Suite, Notion, Buffer

ESCALATION: Flags blockers to Planner (BS Thư unavailable, compliance issue, anomaly)
```

## ⚠️ Critical Success Factors

### Must-Haves (No Substitute)

1. **🎥 BS Thư commitment:** 2 shoots/week minimum (locked schedule)
2. **☑️ Compliance discipline:** 5-item gate on EVERY post (non-negotiable)
3. **📊 Notion updates:** Actual results filled within 24h of posting (data dependency)
4. **⏰ Hypothesis deadlines:** Decisions made on-time (26/5, 31/5) — no deferral
5. **📱 Measurement access:** Meta Business Suite + Buffer read access (for Execution agent)

### Risks & Mitigations

| Risk | Severity | Mitigation |
|---|---|---|
| BS Thư unavailable (illness, schedule) | 🔴 CRITICAL | Backup script library (pre-approved clips) |
| Compliance violation (post goes live) | 🔴 CRITICAL | 5-item gate before publishing (not after) |
| H1/H2 data inconclusive (trend unclear) | 🟠 HIGH | Extended measurement window (7-14 days) |
| Notion not updated (breaks measurement) | 🟠 HIGH | Friday 18:00 deadline (automatic reminder) |
| AI-generated copy feels generic | 🟡 MEDIUM | A/B test against hand-written copy W21 |
| Algorithm change mid-cycle | 🟡 MEDIUM | Weekly Planner synthesis catches shift, pivots fast |
