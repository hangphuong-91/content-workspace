# Platform Specs — Character Limits & Format Rules

## Facebook

| Element | Giới hạn | Tối ưu |
|---|---|---|
| Post caption | 63,206 ký tự | 40–80 từ (before "See more") |
| Dòng đầu tiên (hook) | — | Dưới 125 ký tự |
| Hashtag | Không giới hạn | 3–5 hashtag |
| Video caption | Tương tự | — |
| Ad primary text | 125 ký tự hiển thị | Dưới 125 ký tự cho full display |
| Ad headline | 40 ký tự | 25–30 ký tự |
| Ad description | 30 ký tự | — |

## Instagram

| Element | Giới hạn | Tối ưu |
|---|---|---|
| Caption | 2,200 ký tự | 138–150 ký tự trước "more" |
| Hashtag | 30 hashtag | 5–10 targeted hashtag |
| Bio | 150 ký tự | — |
| Story text | — | Ngắn, dễ đọc nhanh |

## TikTok

| Element | Giới hạn | Tối ưu |
|---|---|---|
| Caption | 2,200 ký tự | 1–2 dòng + hashtag |
| Hashtag | Không giới hạn | 3–5 hashtag |
| Video length | 10 phút | 30–60s (cao nhất reach), 3–7 phút (deep content) |
| Hook (3 giây đầu) | — | Phải hoàn chỉnh ý — người xem decide trong 3s |
| Text overlay | — | Font to, contrast cao, safe zone 150px trên/dưới |

## LinkedIn

| Element | Giới hạn | Tối ưu |
|---|---|---|
| Post | 3,000 ký tự | 1,300 ký tự (before "see more") |
| Article | 120,000 ký tự | 1,000–2,000 từ |
| Headline | 220 ký tự | 120 ký tự |
| Hashtag | — | 3–5 hashtag, cuối bài |
| Video | 10 phút | 30–90s cho organic |

## Email

| Element | Giới hạn | Tối ưu |
|---|---|---|
| Subject line | Không giới hạn | 40–50 ký tự (hiển thị đủ mobile) |
| Preview text | — | 80–100 ký tự |
| Email width | — | 600px (responsive) |
| CTA button | — | 1 button, màu nổi bật, text 3–5 từ |
| P.S. line | — | 1 câu — thường đọc nhiều hơn body |

## YouTube

| Element | Giới hạn | Tối ưu |
|---|---|---|
| Title | 100 ký tự | 60–70 ký tự (hiển thị đủ search) |
| Description | 5,000 ký tự | 200–300 từ đầu (visible before "more") |
| Tags | 500 ký tự | 10–15 tags mix broad + specific |
| Thumbnail text | — | Ít hơn 6 từ, font to, contrast cao |

## Blog / SEO

| Element | Giới hạn | Tối ưu |
|---|---|---|
| Title tag | — | 50–60 ký tự (không bị cắt trên Google) |
| Meta description | — | 150–160 ký tự |
| H1 | — | 1 per page, có keyword chính |
| Bài viết | — | 1,000–2,500 từ (tùy topic) |
| Internal links | — | 2–5 links/bài |
